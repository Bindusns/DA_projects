

# 📈 Coca-Cola Stock Market Analysis (1980–2026)

## 📌 Project Overview

This project analyzes **historical stock price data of The Coca-Cola Company (KO)** from **1980 to 2026**. The analysis was performed using **Python for data processing**, **SQL for querying**, and **Power BI for interactive visualization**.

The objective of this project is to demonstrate **data analysis, visualization, and dashboard development skills** by transforming raw financial data into meaningful insights about stock market trends.

---

## 🛠 Tools & Technologies

This project uses the following tools:

* **Python** – Data cleaning and exploratory data analysis
* **Pandas & Matplotlib** – Data manipulation and visualization
* **SQL** – Data querying and transformation
* **Power BI** – Interactive dashboard creation
* **Jupyter Notebook** – Analysis environment

---

## 📊 Dashboard Features

The Power BI dashboard provides insights into Coca-Cola stock performance through:

* **Average Stock Price KPI**
* **Maximum Closing Price KPI**
* **Total Trading Volume KPI**
* **Stock Price Trend Analysis**
* **Yearly Average Closing Price**
* **Trading Volume Trend**
* **Monthly Stock Returns Table**
* **Interactive Year Filter**

These visualizations help understand **long-term stock behavior and market activity**.
